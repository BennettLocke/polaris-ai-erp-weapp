# Changelog

All notable changes to Polaris AI ERP WeApp are documented here.

## [0.1.0] - 2026-06-02

### Added

- Initial public release foundation for the Polaris AI Agent mini-program frontend.
- uni-app + Vue 3 pages for home, category browsing, product list, product detail, workflow orders, sales orders, profile, settings, policies, and contact.
- API adapters for Polaris AI Agent product, category, home, auth, customer, workflow order, and sales order endpoints.
- Environment-driven backend and customer-service configuration through `.env.example`.
- Unit tests and WeChat mini-program build validation.

### Compatibility

- Requires `polaris-erp-agent >= 0.1.0`.
- Designed to consume mini-program APIs exposed by the Polaris AI Agent server.
